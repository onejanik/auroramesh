# Moderations-Richtlinien (Draft)

- **Community Standards:** Respekt, keine Hate Speech, keine Gewaltverherrlichung.
- **Content Safety:** Nudity, Self-Harm, Spam Policies mit Eskalationsstufen.
- **Strike System:** Verwarnung → Temporäre Sperre → Permanente Sperre.
- **Review Workflow:** User Report → KI-Vorsortierung → Moderator Review → Appeal.
- **Transparenz:** Monatliche Reports, Hinweis auf eingesetzte KI.
